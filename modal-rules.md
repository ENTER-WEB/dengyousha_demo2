# モーダル出現パターン・表示ルール

このドキュメントは、`index.html` / `style.css` / `script.js` に実装しているモーダルの表示条件と挙動をまとめた仕様書です。

## 1. 下層ページ未公開案内モーダル

### 対象モーダル

- ID: `demoModal`
- 用途: デモサイト内の未実装リンク、下層ページ想定リンクをクリックした際に表示する案内

### 表示されるタイミング

以下の条件に該当するリンクをクリックしたときに表示されます。

- `href="#"` が設定されているリンク
- 実際の下層ページURLがまだ設定されていないリンク
- 施工実績、会社情報、SNS、資料DL、問い合わせフォームなど、デモ上では遷移先を用意していない導線

### 表示されないリンク

以下のリンクでは表示されません。

- `href="#top"` など、ページ内アンカーとして実在するリンク
- `href="#business"` など、同一ページ内のセクションへ移動するリンク
- `tel:` など、`#` 以外のリンク

### 表示文言

見出し:

```text
このページはデモサイトのご案内です
```

本文:

```text
こちらはデモサイトのため、現在はTOPページのみ公開しております。

実際の納品時には、ご提案内容や制作範囲に応じて、
TOPページのみの1ページ構成、または下層ページを含む複数ページ構成にて制作いたします。

サービス内容・会社情報・お問い合わせ導線などは、
目的やご予算、運用方法に合わせて最適な形で設計いたします。
```

補足:

```text
※確認用のため、一部リンクはデモルールに基づき表示されています。
```

補足カード1:

```text
構成はご提案内容に応じて調整
1ページ構成・複数ページ構成どちらにも対応可能です。
```

補足カード2:

```text
目的に合わせた設計
お問い合わせや予約、採用応募など、ゴールに合わせて設計します。
```

### 閉じ方

以下の操作で閉じます。

- 右上の閉じるボタン
- 背景クリック
- `閉じる` ボタン
- `TOPページを見る` ボタン
- Escキー

### 再表示ルール

- `href="#"` のリンクをクリックするたびに表示されます。
- セッション内で閉じても、次に未実装リンクをクリックした場合は再表示されます。

## 2. PC表示推奨モーダル

### 対象モーダル

- ID: `spModal`
- 用途: PC未満の画面幅で閲覧したユーザーに、PC表示での確認を促す案内

### 表示されるタイミング

ページ読み込み後、以下の条件に該当する場合に自動表示されます。

- 画面幅が `1023px以下`
- スマートフォン表示
- タブレット表示
- タブレット縦向き表示
- ブラウザ幅をPC未満まで狭めた状態

現在の実装条件:

```javascript
window.matchMedia("(max-width: 1023px)").matches
```

### 表示されないタイミング

以下の場合は表示されません。

- 画面幅が `1024px以上`
- PC表示
- タブレット横向きなどで画面幅が `1024px以上` になっている場合
- 同一ブラウザセッション内ですでに閉じている場合

### 表示文言

見出し:

```text
このデモサイトはPC表示のみとなっております
```

本文:

```text
現在ご覧の画面幅では、実際のデザインと異なって見える場合があります。

恐れ入りますが、PCまたはタブレットの横向き表示でご確認ください。
```

### 閉じ方

以下の操作で閉じます。

- 右上の閉じるボタン
- 背景クリック
- `閉じる` ボタン
- Escキー

### 再表示ルール

このモーダルは同一セッション内で一度閉じると再表示されません。

実装上は `sessionStorage` に以下の値を保存しています。

```javascript
sessionStorage.setItem("spModalClosed", "1");
```

そのため、以下の場合は再度表示されます。

- 新しいタブで開いた場合
- ブラウザを閉じて再度開いた場合
- セッションストレージを削除した場合
- 開発者ツール等で `spModalClosed` を削除した場合

## 3. モーダル背景・視覚表現

### 共通背景

両方のモーダルは同じ背景スタイルを使用しています。

```css
background: rgba(0, 24, 16, 0.48);
backdrop-filter: blur(8px);
```

### デザイン方針

- サイト全体の緑基調に合わせる
- 白背景の角丸ダイアログ
- 情報アイコンは淡いグリーン背景
- ボタンは緑の塗り、または緑のアウトライン
- 参考画像の案内モーダルをベースにしつつ、電業社サイトのトンマナに合わせる
- 他案件へ流用する場合は、制作中サイトのトンマナを参照し、色・ボタン形状・余白・角丸・アイコン表現をそのサイトのデザインに合わせて調整する
- モーダルだけが別デザインに見えないよう、既存の見出し、本文、CTA、カード、背景色のルールを優先して反映する

## 4. 優先順位

### PC表示時

PC幅では、PC表示推奨モーダルは表示されません。

未実装リンクをクリックした場合のみ、下層ページ未公開案内モーダルが表示されます。

### スマートフォン・タブレット表示時

ページ読み込み時に、PC表示推奨モーダルが自動表示されます。

その後、未実装リンクをクリックした場合は、下層ページ未公開案内モーダルが表示されます。

### 同時表示について

通常操作では、2つのモーダルが同時に表示される想定はありません。

ただし、PC表示推奨モーダルを閉じずに未実装リンクをクリックするような特殊操作をした場合は、後から開いたモーダルが上に重なる可能性があります。
